
## рҹ”’ `web/src/middleware.ts`

Intercepts every incoming request before it reaches a page or API route.

- Reads the session cookie and validates the JWT token.
- Redirects unauthenticated users away from protected routes (e.g., `/admin`, `/agent`).
- Allows public routes (`/`, `/properties`, `/login`, `/signup`) to pass through freely.
